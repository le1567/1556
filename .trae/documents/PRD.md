# 产品需求文档（PRD）

## 项目概述

打造一款单文件 HTML 的奢华信息中枢，将“电视直播”与“实时天气”融于一体。页面应呈现未来豪宅私人影院的沉浸感：全息气象仪、黑曜石电视墙、流动极光与星尘粒子，构成一件可交互的数字艺术品。

## 目标用户

追求视觉品质与信息整合体验的桌面/移动端用户，期望在一个无滚动、全屏沉浸的界面中同时获取直播内容与本地天气。

## 核心功能

1. **电视直播播放器**
   - 使用 hls.js 播放公开 M3U8 直播源，自动降级到原生 HLS 支持。
   - 内置至少 8 个国内主流电视频道，支持底部频道栏切换。
   - 切换时带“关屏-切换-开屏”的电影级转场动画与星光粒子反馈。
   - 极简悬浮控制条：播放/暂停、音量滑块。

2. **实时天气展示**
   - 集成 OpenWeatherMap 免费 API。
   - 默认城市：北京；支持点击齿轮输入城市名（含中文）并回车刷新。
   - 显示当前温度、体感温度、天气描述、湿度、风速。
   - 展示未来 3 天迷你预报。
   - 根据天气类型渲染动态矢量场景（晴、多云、雨、雪、雾）。

3. **动态环境**
   - Canvas 粒子宇宙背景，粒子随天气改变运动形态与色温。
   - 根据真实时间微调全局环境光色调。
   - 顶部偶现漂移极光光幕。
   - 鼠标聚光灯跟随效果。

## 视觉方向

- **风格**：深邃奢华黑暗主题，现代豪华影音室 + 全息气象站。
- **主色**：黑、深蓝、金、玫瑰金。
- **材质**：磨砂玻璃、液态金丝边框、水晶棱镜、呼吸辉光。
- **字体**：标题使用 'Cormorant Garamond' / 'Playfair Display'，正文使用 'Inter' / 'Montserrat'。
- **动效**：贝塞尔曲线过渡、多层视差、动态模糊、光尘粒子。

## 页面布局

- 满屏沉浸，无滚动条。
- 桌面端：左侧 35% 悬浮全息天气卡片，右侧 65% 电视播放器窗口，底部中央频道栏，左上角 Logo，右下角时间水晶球。
- 移动端：上下堆叠，天气卡片在上精简显示，播放器在下，频道栏为横向滑动药丸列表。

## 技术约束

- 单文件 HTML，包含全部 CSS/JS 与 CDN 引用。
- 使用 hls.js、Google Fonts。
- 天气图标与特效全部代码实现，不使用图片资源。
- API KEY 集中在脚本顶部配置。
- 频道源集中配置为数组。
- 处理 API 错误，不影响直播播放。
- 响应式设计，移动端减少粒子数量保证性能。

## 成功标准

- 打开页面即进入沉浸式影院氛围。
- 频道切换流畅，直播可正常播放（依赖公开源可用性）。
- 天气数据正确展示，错误时给出优雅提示。
- 所有动效在桌面端流畅，移动端不卡顿。
